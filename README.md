# Victrix Capital - Website Institucional

Site institucional da Victrix Capital, agente autônomo de investimentos parceiro XP Investimentos.

## 🎨 Identidade Visual

- **Cores principais:**
  - Deep Forest: `#0E1C0E`
  - Lime Green: `#88E833`
  - Mid Green: `#2E6F3A`
  - Sage: `#D5DAD0`

- **Tipografia:** Zalando Sans SemiExpanded

## 📁 Estrutura do Projeto

```
victrix-website/
├── assets/
│   ├── css/         # Estilos
│   ├── js/          # Scripts
│   ├── images/      # Imagens e logos
│   └── fonts/       # Fontes
├── index.html       # Página inicial
├── sobre.html       # Sobre nós
├── servicos.html    # Serviços
└── contato.html     # Contato
```

## 🚀 Como usar

1. Clone o repositório
2. Abra `index.html` no navegador
3. Ou use um servidor local:
   ```bash
   python -m http.server 8000
   # ou
   npx serve
   ```

## 🌐 Deploy

Este site pode ser hospedado em:
- GitHub Pages
- Vercel
- Netlify
- Qualquer servidor web estático
